# Blockchain_lab4
Bank Smart Contract
Функциональность контракта

Функции лабораторной работы:

    Вывод средств владельцем - ownerWithdrawal(uint _value)
    Выдача VIP статуса - setVIPStatus(address _customer)
    Начисление процентов - applyInterest()
    Оплата за услуги - payForService(uint _amount)

Структуры данных
Statuses enum:

    VIP - VIP статус клиента
    General - обычный статус клиента

Customer struct:

    name - имя клиента
    customerBalance - баланс клиента
    status - статус (VIP/General)
    registrationTime - время регистрации
    lastDepositTime - время последнего депозита

Модификаторы доступа

    public - owner - адрес владельца контракта
    external - все функции взаимодействия
    view - функции чтения данных

Математические расчеты

    7% годовых
    Расчет на основе времени с последнего депозита
    Автоматическое обновление времени последней операции

Безопасность

    Проверка регистрации клиента
    Проверка достаточности баланса
    Проверка прав владельца
    Проверка корректности сумм
    Только владелец может назначать VIP статус
    Проценты начисляются по требованию клиента
    Оплата услуг списывается с баланса клиента
    Время измеряется в секундах (block.timestamp)
